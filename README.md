# Secure Chat Application

A real-time secure chat application with video/audio calls, file sharing, and group messaging.

## Features
- 🔐 User Authentication
- 💬 Public, Private & Group Chats
- 📹 Video Calling
- 📞 Audio Calling
- 📎 File Sharing
- 🎤 Voice Messages
- 🖼 Image Preview
- 👥 Group Management

## Tech Stack
- Python
- Socket Programming
- Tkinter GUI
- OpenCV (Video)
- PyAudio (Voice)

## How to Run Locally
1. Clone: `git clone https://github.com/FizzaFaisal07/secure-chat-application`
2. Install: `pip install -r requirements.txt`
3. Run server: `python server.py`
4. Run client: `python client.py`
5. Login with: alice/12345, bob/456789
